# PWA Boilerplate and Sample

## Prerequisite

- Node ^14.0.0

## Contains

- [x] [Typescript](https://www.typescriptlang.org/) ~4.0.0
- [x] [Angular](https://github.com/angular) ^10.0.0

## Installation & Running

```
$ npm install

$ npm start
```
- Check project at port `4100`
