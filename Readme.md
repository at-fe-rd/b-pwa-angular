# PWA Boilerplate and Sample

## Prerequisite

- Node ^10.9.0

## Contains

- [x] [Typescript](https://www.typescriptlang.org/) 3.4.5
- [x] [Angular](https://github.com/angular) 8.0.0

## Installation & Running

```
$ npm install

$ npm start
```
- Check project at port `4100`
